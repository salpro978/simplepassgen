# Simple Password Generator

SimplePassGen adalah program sederhana untuk membuat password secara random.

Project ini dibuat untuk tugas GSLC DevOps.

## Cara Install

Clone repository ini ke komputer:

```
git clone https://github.com/mybday123/simplepassgen.git
```

Masuk ke folder project:

```
cd simplepassgen
```

## Cara Menjalankan Program

Jika menggunakan Python:

```
python main.py
```

## Menjalankan Menggunakan Docker

Build docker image:

```
docker build -t simplepassgen .
```

Jalankan container:

```
docker run simplepassgen
```
